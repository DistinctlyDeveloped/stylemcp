Frequently Asked Questions

What's the difference between basic and AI rewrites?

Basic rewrites (Free tier): Simple word swaps based on your vocabulary rules (e.g., "utilize" → "use"). Fast and predictable.

AI rewrites (Pro+): Intelligent rewrites using Claude AI that understand context, fix complex patterns, and maintain your brand voice naturally.

What counts as a request?

Each API call to /api/validate, /api/rewrite, or /api/validate/batch counts as one request. AI rewrites count as 1 request. Batch requests count based on the number of items processed.

Can I try before signing up?

Yes! Use our live demo to test validation without creating an account. You get 10 free requests per hour, no signup required.

What happens if I exceed my limit?

API requests will return a 429 error until your quota resets on the 1st of each month. You can upgrade anytime to get more requests immediately.

Can I self-host StyleMCP?

The core validation engine is open source (MIT). Enterprise customers can self-host with AI features included. View on GitHub

How is this different from Writer or Jasper?

Writer and Jasper are content generators with brand features built in. StyleMCP is brand validation—it works with ANY AI (Claude, GPT, Llama, your own models) and validates/rewrites their output. We're the quality layer, not the generator.
